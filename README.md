# JsSimpleBotzz

Minecraft Bot Manager written in Rust with Node.js integration and GUI.

---

## English

### Requirements

- [Rust](https://www.rust-lang.org/tools/install) (1.70+)
- [Node.js](https://nodejs.org/) (18+)

### Build and Run

**Linux GUI:**

```bash
cd platforms/linux
cargo run --release
```

**Windows GUI:**

```cmd
cd platforms\windows
cargo run --release
```

**CLI Version:**

```bash
cargo run --release --bin js_simple_botzz
```

**Node.js Integration:**

```bash
cd nodejs
npm install
npm start
```
